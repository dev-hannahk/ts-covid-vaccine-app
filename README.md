## COVID-19 Vaccinations

### Tech Stack

- TypeScript
- React
- Redux
- Redux-Thunk
- Axios
- Styled-Components
- Highcharts
- React-Datepicker

### Preview

### Goal

- [공공데이터포탈 API](https://www.data.go.kr/index.do)를 활용한 코로나 백신 접종 관련 웹앱 만들기
- 쿼리스트링으로 시도 및 기간(1주, 1개월, 3개월, 전체기간) 필터링
- 데이트피커 라이브러리를 이용해서 직접 입력한 기간의 데이터 조회
- 하이차트 라이브러리를 이용해서 누적 접종률, 일일 접종률 시계열 차트 구현
